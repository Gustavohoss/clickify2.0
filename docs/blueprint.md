# **App Name**: LaunchPad AI

## Core Features:

- AI-Powered Product Creation: Generate digital product ideas and structures, including course outlines and assets, using AI tools. The LLM uses the data stored in Lovable Cloud, acting as a tool, to accomplish its job.
- Automated Funnel Builder: Create complete marketing funnels, including landing pages, email sequences, sales pages, and checkout processes, automatically.
- TikTok Content Generator: Generate TikTok profile content, including bio, video ideas, scripts, and posting schedules, leveraging AI-driven suggestions. The LLM uses the templates stored in Lovable Cloud, acting as a tool, to accomplish its job.
- Landing Page Builder: Design and customize responsive landing pages with a drag-and-drop interface, optimized for different devices.
- Pricing and Plan Comparison: Display a clear pricing table with feature comparisons for Free, Pro, and Premium plans, highlighting the most popular option.
- User Authentication and Authorization: Secure user accounts with email/password login and optional Google OAuth integration, managed through Lovable Cloud.
- Template and asset storage: Store templates, models and other digital assets into Lovable Cloud storage.

## Style Guidelines:

- Primary color: Electric blue (#0EA5FF) to convey energy and confidence.
- Accent color: Deep purple (#7C3AED) for a futuristic and authoritative feel.
- Background: Deep black (#0B0F13) with subtle gradients from black to a soft purple for depth.
- H1: 'Inter', a sans-serif font, large, condensed, and weight 700 for a futuristic/impactful feel.
- Body: 'Inter', a sans-serif font, geometric and legible with a line-height of ≥ 1.5.
- 8pt grid system with responsive widths, container max width of 1200–1400px.
- Subtle fade and translateY (200–300ms, ease-in-out) animations for transitions.
